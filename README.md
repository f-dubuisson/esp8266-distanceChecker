# esp8266-blindController

Program for the ESP8266 to remotely check a distance using a HC-SR04.

## How to install

- Install PlatformIO
- Import the project
- Rename 'src/config-sample.h' as 'src/config.h' and set the right values
- Build and upload

## Commands

Format is:

    http://\<device ip\>

## Hardware

- ESP8266 (I use a NodeMCU device)
- HC-SR04
